Modular Arithmetic

It involves dealing with distribution on % sign over operations.

quicklist
  1. (a+b)%MOD = (a%MOD + b%MOD)%MOD
  2. (a-b)%MOD = (a%MOD - b%MOD + MOD)%MOD
  3. (a*b)%MOD = (a%MOD * b%MOD)%MOD
Division involves a special technique discussed as follows,
say we are given a, b. We are required to find (a/b)%MOD.
1. It is defined iff a%b == 0,
2. In such case:
(a/b)%MOD = (a%MOD * i%MOD)%MOD

where, 'i' is the Modular inverse of b,
it is calculated using fermat's little theorem post below.



Binary Exponenciation

Although basic pow(x,y) is already implemented in most of the standard libraries. calculating xy is very important for many questions in log( y ) time complexity. So following is the implementation. 


ll binexpy(ll x, ll y)
{
	if(y == 0)
	{
		return 1;
}
	else if(y%2==0)
	{
		return binexpy(x*x,y/2);
	}
	else if(y%2==1)
	{
		return x*binexpy(x*x,y/2);
	}
}
Most of the times xy will exceed 1018 so we are expected to print the answer modulo some prime number. some of these prime numbers are :
1000000007 ( 109 + 7 ) 1000003 ( 106 + 3 )

So we are now led to implementing binary exponentiation with MOD( required prime number for modulus operation ) 

ll binexpymod(ll x,ll n,ll mod)
{
	ll prod = 1;
	if(n==0)
	{
		return 1;
	}
	else if(n%2==0)
	{
		prod = ((x%mod)*(x%mod))%mod;
		return binexpymod(prod,n/2,mod);
	}
	else{
		prod = ((x%mod)*(x%mod))%mod;
		return ((x%mod)*binexpymod(prod,(n-1)/2,mod))%mod;
	}
}



Fermat's Little Theorem

Simply stated Fermat's little theorem :
ap-1 = 1(mod p)
Note: p-1 comes from PHI(p) = p-1 for primes

Now to implement it we will use binexpymod as defined above:
ans = binexpymod(a,MOD-2,MOD);
where, ans = a-1(% MOD), Note (% MOD) = (mod p)

* Quick explanation:
* from FLT: ap-1 = 1(mod p)
* hence, ap-2 = a-1(mod p)

Prime Factorization

Prime factorization of 12:
12 : 2 X 2 X 3

19 : 19
Note: 1 is not a prime

Implementation as follows: O ( root(N) ) 

void primeFactors(ll n)
{
	vector < ll > temp;
    while (n % 2 == 0)
    {
        temp.pb(2);
        n = n/2;
    }
    for (int i = 3; i <= sqrt(n); i = i + 2)
    {
        while (n % i == 0)
        {
            temp.pb(i);
            n = n/i;
        }
    }
    if (n > 2)
        temp.pb(n);

	//do what you want with vector
}
Now sometimes for query based questions we may require O( log N ) processing, so let's see what we got.
Implementation in preprocessing: O(N log N), queries O( log a )

const ll N = 1e7; // check this please reduce to 5,000,005 if required
//prime factors in O(log n)
int primechk[N];
int fprime[N];
void preprocess(void)
{
    int i, j;
    for(i=2 ; i< N ; i++)
    {
        primechk[i]=1;
    }
    for(i=2 ; i< N ; i++)
    {
        if(primechk[i]==1)
        {
            fprime[i]=i;
            for(j=2 ; i*j< N ; j++)
            {
                primechk[i*j]=0;
                if(fprime[i*j]==0)
                {
                    fprime[i*j]=i;
                }
            }
        }
    }
}
/**
 * Prime factorization in log n using sieve
**/
vector < ll > primefactlogn(ll n)
{
	vector < ll > temp;
	while(n!=1)
    {
        temp.pb(fprime[n]);
        n=n/fprime[n];
    }
    return temp;
}



divisors

Divisors of 12:
12 : 1, 12, 2, 6, 3, 4

19 : 1, 19
Note: Order can be sorted

Implementation as follows: O ( root(N) ) 

vector divisors(ll x)//O( sqrt(n) )
{
	vector < ll > divs;
	for(int i=1;i*i<=x;i++)
	{
		if(x%i == 0)
		{
			if(i*i == x)
			{
				divs.pb(i);
			}
			else
			{
				divs.pb(i);
				divs.pb(x/i);
			}
		}
	}
	return divs;
}//sort if you want

EulerPHI function

PHI(n) = numbers of coprimes in 1-n for n
Note: for prime p PHI(p) = p-1 


// sample program to calculate PHI(n) for x
ll phi(ll x) 
{ 
    ll ans = x;
    for (int p = 2; p * p <= x; p++) {  
        if (x % p == 0) { 
              
            while (x % p == 0) 
                x /= p; 
            ans -= ans / p; 
        } 
    } 
    if (x > 1) 
        ans -= (ans / x); 
    return ans; 
}
One Question that can be solved using PHI is:
Q) Given integer x, calculate ∑( LCM(i,n) ), for i in range(1,N]
answer to query is n*[1+∑(PHI(i)])/2 ,
Note: ∑(PHI(i)) = summation PHI[i] for i in range(1,n]
